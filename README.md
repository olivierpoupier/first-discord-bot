# discord reaction bot
A simple bot in Node using TS, based on the tutorial ['Working with TypeScript, Dependency Injection, and Discord Bots'](https://www.toptal.com/typescript/dependency-injection-discord-bot-tutorial) by Micha≈Ç Krakiewicz

# Installing

1. `npm install`
2. Replace your Discord Bot token to the `.env` file

# Running

## CLI

Compiling the project in watch mode: `npm run watch`

Running the application: `npm start`

Running the tests: `npm test`
